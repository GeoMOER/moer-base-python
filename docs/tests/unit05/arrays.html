Arrays - Test

1. Why are arrays used in Python?

A) To store multiple values in a single variable
B) To improve speed of calculations
C) Both A and B
Correct answer: C) Both A and B

2. How do you create a NumPy array from a list my_list = [1, 2, 3]?

A) arr = np.array(my_list)
B) arr = array(my_list)
C) arr = np.create(my_list)
Correct answer: A) arr = np.array(my_list)

3. How can you check the data type of a NumPy array arr?

A) arr.dtype
B) type(arr)
C) arr.data_type()
Correct answer: A) arr.dtype

4. How do you find the length of a NumPy array arr?

A) len(arr)
B) arr.length()
C) arr.size
Correct answer: A) len(arr)

5. How do you add an element 4 to a NumPy array arr?

A) arr.add(4)
B) arr.append(4)
C) np.append(arr, 4)
Correct answer: C) np.append(arr, 4)

6. How can you perform an operation (addition) on two arrays arr1 and arr2 of the same length?

A) result = arr1 + arr2
B) result = np.add(arr1, arr2)
C) Both A and B
Correct answer: C) Both A and B
