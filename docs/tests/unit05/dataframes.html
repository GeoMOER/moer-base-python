DataFrames - Test

1. How do you define a DataFrame from multiple lists in Python?

A) df = DataFrame(list1, list2)
B) df = pd.DataFrame({'col1': list1, 'col2': list2})
C) df = DataFrame({'col1', list1, 'col2', list2})
Correct answer: B) df = pd.DataFrame({'col1': list1, 'col2': list2})

2. How do you access a specific column named 'Age' in a DataFrame df?

A) df.{Age}
B) df['Age']
C) df(Age)
Correct answer: B) df['Age']

3. How do you access the first row of a DataFrame df?

A) df[0]
B) df.iloc[0]
C) df.first()
Correct answer: B) df.iloc[0]

4. How do you add a new column named 'Salary' to a DataFrame df with a value of 50000?

A) df['Salary'] = 50000
B) df.add_column('Salary', 50000)
C) df.append('Salary', 50000)
Correct answer: A) df['Salary'] = 50000

5. How do you check the shape of a DataFrame df?

A) df.shape()
B) df.shape
C) shape(df)
Correct answer: B) df.shape

6. How do you rename a column 'OldName' to 'NewName' in a DataFrame df?

A) df.rename(columns={'OldName': 'NewName'})
B) df.rename('OldName' = 'NewName')
C) df.change_column('OldName', 'NewName')
Correct answer: A) df.rename(columns={'OldName': 'NewName'})
