### Zweck der Datenverarbeitung

Diese Webseite dient ausschließlich dem **Lehrzweck** im Rahmen eines Python-Programmierkurses an der Philipps-Universität Marburg.  
Die Seite wird **nicht für kommerzielle Zwecke** genutzt und ist **ohne Login** zugänglich.

**Es werden keine personenbezogenen Daten erfasst, gespeichert oder verarbeitet.**  
Die Nutzung der Webseite ist **vollständig anonym**.


### Erhebung und Verarbeitung personenbezogener Daten

Beim Besuch dieser Webseite werden durch die Philipps-Universität Marburg selbst keine personenbezogenen Daten aktiv erhoben (z. B. durch Formulare, Tracking oder Benutzerkonten).
Die Webseite erfordert **kein Login **.
Allerdings wird die Webseite über GitHub Pages bereitgestellt. Dabei kann es zur Verarbeitung personenbezogener Daten durch GitHub kommen.


### Hosting über GitHub Pages

Diese Webseite wird über den Dienst GitHub Pages der GitHub bereitgestellt.
Beim Zugriff auf die Webseite werden durch GitHub automatisch technische Zugriffsdaten verarbeitet, insbesondere:

IP-Adresse
Datum und Uhrzeit des Zugriffs
Browsertyp und -version
Betriebssystem
Referrer-URL

Diese Daten werden von GitHub in Server-Logdateien gespeichert.
Die Verarbeitung erfolgt durch GitHub als eigener Verantwortlicher und nicht im Auftrag der Philipps-Universität Marburg.


### Drittlandübermittlung

GitHub gehört zur Microsoft und verarbeitet Daten teilweise auf Servern außerhalb der Europäischen Union, insbesondere in den USA.
Dabei kann nicht ausgeschlossen werden, dass personenbezogene Daten in ein Drittland übermittelt werden, in dem kein gleichwertiges Datenschutzniveau wie in der EU besteht.
Weitere Informationen finden Sie in der Datenschutzerklärung von GitHub.
