If and elif Statements - Test

1. What is the purpose of the `elif` statement?

A) To specify another condition to check if the previous `if` condition was False.
B) To close an `if` statement.
C) To negate a previous condition.
Correct answer: A) To specify another condition to check if the previous `if` condition was False.

2. How many `elif` statements can you use in an `if` block?

A) Only 1
B) As many as needed
C) None, `elif` does not exist in Python
Correct answer: B) As many as needed

3. What is the output of the following code?

x = 50 if x < 10:     print("Less than 10") elif x > 15:     print("Greater than 15") elif x == 50:     print("Equal to 50") else:     print("None of the above")
A) Less than 10
B) Greater than 15
C) Equal to 50
D) None of the above
Correct answer: B) Greater than 15

4. What is the difference between `else` and `elif`?

A) `else` checks for a condition, while `elif` does not.
B) `elif` checks for a condition, while `else` is used when none of the previous conditions are met.
C) `elif` and `else` are functionally the same.
Correct answer: B) `elif` checks for a condition, while `else` is used when none of the previous conditions are met.

5. What is the output of the following code?

x = 50 if x < 25:     print("Less than 25") elif x < 50:     print("Less than 50") elif x == 50:     print("Equal to 50") else:     print("Greater than 50")
A) Less than 25
B) Less than 50
C) Equal to 50
D) Greater than 50
Correct answer: C) Equal to 50

6. Small coding task: Write an if-elif-else block that checks if a number is negative, zero, or positive. What will the following code print for y = -5?

y = -5 if y > 0:     print("Positive") elif y == 0:     print("Zero") else:     print("Negative")
A) Positive
B) Zero
C) Negative
Correct answer: C) Negative
