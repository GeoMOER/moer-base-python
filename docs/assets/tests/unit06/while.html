While Loops - Test

1. What is a `while` loop used for in Python?

A) Iterating over a sequence
B) Repeating code while a condition is true
C) Checking conditions once
Correct answer: B) Repeating code while a condition is true

2. What is the result of the following code?

x = 3 while x > 0:     print(x)     x -= 1
A) 3 2 1
B) 2 1 0
C) 3 2
Correct answer: A) 3 2 1

3. How do you ensure a `while` loop does not run infinitely?

A) Call `break` immediately
B) Use the `pass` statement
C) Ensure the loop has a terminating condition
Correct answer: C) Ensure the loop has a terminating condition

4. What keyword is used to exit a loop?

A) `exit`
B) `break`
C) `return`
Correct answer: B) `break`

5. What is the output of the following code?

x = 5 while x > 0:     print(x)     if x == 3:         break     x -= 1
A) 5 4
B) 5 4 3
C) 5 4 3 2 1
Correct answer: A) 5 4

6. How do you skip to the next iteration in a `while` loop?

A) `skip`
B) `continue`
C) `break`
Correct answer: B) `continue`

7. What happens if the condition in a `while` loop is always `True`?

A) The loop runs infinitely
B) The loop runs once
C) The loop stops immediately
Correct answer: A) The loop runs infinitely

8. How do you create a loop that runs forever?

A) `while True:`
B) `while 1:`
C) Both
Correct answer: C) Both
