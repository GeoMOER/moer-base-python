Booleans - Test

1. What are the two possible Boolean values in Python?

A) True and False
B) Yes and No
C) 1 and 0
Correct answer: A) True and False

2. Which operator is used to compare two values for equality in Python?

A) ==
B) =
C) !=
Correct answer: A) ==

3. What does the following expression return: 5 > 3?

A) True
B) False
C) None
Correct answer: A) True
