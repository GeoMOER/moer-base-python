Strings - Test

1. How do you define a string in Python?

A) my_string = 'hello'
B) my_string = hello
C) my_string = {hello}
Correct answer: A) my_string = 'hello'

2. What does my_string[0] return if my_string = 'hello'?

A) h
B) e
C) o
Correct answer: A) h

3. Which method is used to replace all occurrences of a substring?

A) replace()
B) find()
C) split()
Correct answer: A) replace()

4. What does the len() function return for a string?

A) The number of words
B) The number of characters
C) The number of lines
Correct answer: B) The number of characters

5. What is the result of 'hello' * 3?

A) hellohellohello
B) hello3
C) hello * 3
Correct answer: A) hellohellohello

6. How can you check if a string contains a specific substring?

A) Using the find() method
B) Using the in keyword
C) Both A and B
Correct answer: C) Both A and B
