Integers and Floats - Test

1. What is the result of int(3.7) in Python?

A) 4
B) 3
C) 3.7
Correct answer: B) 3

2. What function would you use to convert an integer to a float?

A) int()
B) str()
C) float()
Correct answer: C) float()

3. What is the output of print(type(10.5)) in Python?

A) <class 'int'>
B) <class 'float'>
C) <class 'str'>
Correct answer: B) <class 'float'>

4. Which of the following is NOT a valid float value in Python?

A) 3.14
B) -1.0
C) 2,718
Correct answer: C) 2,718

5. What is the result of float(5)?

A) 5.0
B) 5
C) 0.5
Correct answer: A) 5.0

6. What is the result of dividing two integers, like 10 / 2?

A) 5.0
B) 5
C) 10.0
Correct answer: A) 5.0
